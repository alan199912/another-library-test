Las alertas mantienen a los usuarios informados sobre el estado del sistema y puede requerir o no que realicen una acción.

Tipos de alertas

Las opciones de alerta muestran su color semántico e ícono correspondiente para ayudar a las personas con deficiencia de visión del color a discernir el tono del mensaje.

Alertas de error

Se usan para mostrar un error de validación, falla del sistema, o para transmitir algo que debe reconocerse o abordarse de inmediato para continuar con su navegación.

Alertas de éxito

Se usan para informar al usuario sobre una tarea que se realizó con éxito.

Alertas de información

Notifican al usuario sobre un cambio, para darles consejos o brindar información específica sobre la sección en la que se encuentran.

Alertas de advertencia

Esta variante se utiliza para informar sobre una situación que debe abordarse pronto.

Diseño y estructura (Contexto de uso)

En cuerpo de página

Los carteles de alerta están destinados a llamar la atención y pedir a los usuarios que tomen medidas necesarias en el contexto de una página. Se colocan al principio de aquello que se quiera advertir, alineados al margen izquierdo.
