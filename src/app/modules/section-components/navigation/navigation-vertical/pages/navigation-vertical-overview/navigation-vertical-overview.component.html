La navegación permite jerarquizar elementos al margen izquierdo de una página en forma vertical, o mostrar la sub navegación dentro de una sección de un sitio web de forma horizontal.

Navegación Vertical

Se utilizan para brindarle a los usuarios una navegacion ordenada, mostrando enlaces para redireccionamiento organizados de manera vertical incluyendo sub elementos. Generalmente los podemos encontrar en el aside.

Diseño y estructura (Contexto de uso)

Solo la navegación vertical tiene hasta 3 niveles de navegación.
