Las tarjetas agrupan información en contenedores flexibles y actúan como punto de entrada que permite introducir un tema en específico.

Con contenedor - Horizontal - Chica

Se utilizan para acceder a información y dependen de un encabezado.

Con contenedor - Horizontal - Mediana

Se utilizan para acceder a información y dependen de un encabezado.

Con contenedor - Horizontal 6 Chica

Diseño y estructura (Contexto de uso)
