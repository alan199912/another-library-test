Los modales son utilizados como ventanas emergentes para resaltar contenido importante en una página haciendo foco en una sola tarea o mensaje. También bloquean otras interacciones para los usuarios.

Tipos de modales

Modal de confirmación

Cuando un usuario necesita información adicional o contexto para tomar la decisión de qué acción realizar.

Modal de peligro

Cuando un usuario necesita confirmar una acción que afectará sus datos o experiencia de una manera potencialmente negativa, como eliminar archivos, descartar un pago, cancelar un formulario, etc.

Diseño y estructura (Contexto de uso)

Un modal deshabilita el contenido de la página y enfoca la atención del usuario en una sola tarea o mensaje.
