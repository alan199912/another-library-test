El componente Galería de imágenes permite mostrar una colección de imágenes relacionadas en una página.

Tipos de Galería

Las listas de imágenes muestran una colección de imágenes en un grilla organizado. Ayudan a mejorar la comprensión visual del contenido que manejan.

Galería - Interactiva

Use la galería interactiva para recorrer diferentes imágenes.

Galería - Grilla

El componente tomará un estilo de grilla determinado por la cantidad de imágenes que se le asigne mostrar.

Diseño y estructura (Contexto de uso)

En cuerpo de página

Es un recurso muy útil cuando necesitas mostrar una serie de fotos en una página o artículo de tu sitio web. Te permite agruparlas y mostrarlas de una forma más profesionales
