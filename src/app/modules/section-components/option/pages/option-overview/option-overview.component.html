Las opciones aparecen al interactuar con un elemento como un botón o un icono o un selector de formulario. Es una forma simple y compacta de mostrar múltiples opciones.

Tipos de Opciones

Cuando el sistema ya posee los datos que se precisan seleccionar y la persona solo debe optar por uno o varios de ellos. Se muestran al accionar un Desplegable.
